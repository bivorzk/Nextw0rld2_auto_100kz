# Nextw0rld2_auto_100kz

## Overview

**Nextw0rld2_auto_100kz** is a Python project that automatically activates 'karzium', so users do not need to click them manually.

## Features

- Automatic activation of 'karzium's without manual clicks
- 100% Python implementation for easy customization and integration

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/bivorzk/Nextw0rld2_auto_100kz.git
   ```

2. **Run the script:**
   - (If you have python installed you can run it by opening cmd to project folder and typing `python main.py`)

## Requirements

- Python 3.x
- pyautogui
  ```
   pip install cv2
   pip install pyautogui
   pip install numpy
  ``
